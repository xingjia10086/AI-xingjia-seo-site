秒杀官方原版几条街的wechat for Mac版本微信剧透..
================================

原创 星佳 星佳 [星佳是个小人物](javascript:void\(0\);)

在小说阅读器中沉浸阅读

相信用Mac系统的朋友们在使用官方的桌面版微信多多少少会真的很不爽，都会觉得和Wechat for win的功能差距太大（快一年都不更新新功能是什么情况...），不知道是不是确实mac系统下用桌面版微信的用户量太少得不到官方的足够关注，还是压根国内做App的始终不会第一时间投入大量精力到Mac OS上的开发呢？

  

于是，看到AppStore里一大堆的吐槽，而官方版本的更新日期依然停留在2015年8月：

  

![](http://mmbiz.qpic.cn/mmbiz/tOFSAibJ5wqukj2FLkhTcibQ5G9BYregbgDX166QXC00gm7lD5fwLemYXibW9THN8F7tibRQTsftzbVFdWEcVibBQxA/0?wx_fmt=png)  

  

好吧，既然官方对于Mac端如此不重视，作为极客精神的我们，何不动手开发一个呢，于是，在极客玩家中，终于在Github上（全球最大的社交编程及代码托管网站）来自上海的一位大神经过业余时间的开发，终于鼓捣出了一个甚至比官方的wechat for win版本的微信都cool的wechat for mac，接下来看看星佳来揭秘到底有多好用.....  (此处特别感谢@杨震凯 分享信息来源)

  

![](http://mmbiz.qpic.cn/mmbiz/tOFSAibJ5wqukj2FLkhTcibQ5G9BYregbgxOmthrxXg6IrDhCH1ZrKrCDzjmV3LiaKWybdib1WqXwJ0z3OlkdQ8iaOg/0?wx_fmt=png)  

看看星佳琢磨出来的几个超级好用的功能：

1.  Mac桌面端可以直接查看公众号文章，并且方便的分享到各种社交媒体
    
2.  可以直接转发聊天内容给多人哦，聊天内容支持光标选取和部分引用
    
3.  懒得写了，总之真的很牛逼...
    

  

总的来说，绝对是用Mac OS的人一大福利了（Win下的App也在逐步完善中了），但是由于作者只是个人纯技术爱好，加上程序员世界特有的开源精神，所以并不希望被太多人知道导致引起官方的关注而项目流产，由此可能带来的法律问题和被官方禁止继续代码封装等问题。

  

其实从技术上来讲，这种通过封装网页版程序成一个可桌面级别运行的App实在很有趣的事情，感谢知乎大神们贡献的干货：

  

> 如果装有chrome，用命令chrome--app=(URL) 就可以打开一个app风格的网页，然后再做一个指向这个命令的快捷方式，你的目的就达到了。
> 
>   
> 
> Awesome-Electron 中有一大票的开源项目做的都是类似的事情，比如  
> 
> *   Caprine - Unofficial Facebook Messenger app.
>     
> *   Inbox - Unofficial Google Inbox app.
>     
> *   MediumDesk - Unofficial Medium app.
>     
> 
>   
> 都是在 Electron 里直接载官方的 web 应用，然后注入一些代码来“增强桌面体验”，有点类似 Chrome 插件的做法。实话说我觉得挺囧的，官方 web 应用不小心更新一下你的注入代码可能就挂了。不过总归是有这样的需求。  
>   
>   
> 
> > 这么做是否存在法律方面的问题？
> 
> 个人认为很多这类应用使用的 Icon 和产品名字或多或少都有侵权嫌疑，除非有得到授权或者公司本身声明过标志的开放使用。  
> 这类项目很多都会写个 Disclaimer 来说明一下，我也不知道这种声明有没有用。不过毕竟都是非商业用途产品，如果不干什么奇怪的事情的话，公司一般也懒得管？  
>   
>   
> 另外，NW.js 也是一个类似的解决方案，对于打个包来说门槛还更低点。  
> BTW，著名移动端打包器 https://cordova.apache.org/ 最近有了 OSX 的支持，估计离支持 Linux 也不远了。
> 
>

多用飞书，少看微信！
==========

原创 星佳&AI助手 星佳&AI助手 [星佳是个小人物](javascript:void\(0\);)

在小说阅读器中沉浸阅读

自从今年过年，终于有大量的整块的时间把自己关在书房里研究人工智能，突然发现用电脑和用手机的时间越来越多，因为电脑和手机上可以方便地看到屏幕使用时长。

  

意外的发现，终于手机上有一个比微信使用时长更多的软件，那就是飞书。

  

每天已经离不开飞书了，基本我会把我看到的图片，文章甚至工作安排都交给飞书，因为飞书的背后是一个叫做 openclaw 大龙虾的人工智能助理。

  

虽然之前研究过自己在 Windows 和 Mac 系统上搭建，但是各种原因不管是对接海外的 API 还是国内的大模型，总会把精力花在各种解决 bug 上，实际上这是最不应该投入时间的地方。

  

于是在 Kimi 一个月 200 块付费后，用了一个 beta 版的大龙虾，之后和书机器人配置成功，给了机器人一些必备的权限，于是大龙虾真的成为了手机端的私人AI 助理。

  

因为飞书自带了云文档和多维表格的功能，甚至还有任务看板，也就是说，只要你通过语音传达正确的指令给你的机器人，并且确定你的机器人安装了对应的 skills，那么你的机器人拥有相关的权限后，真的可以自己建表格，自己填内容，自己去整理每天和你的沟通，还有梳理各种任务进度。

  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/7SRWibfxWe0GoTQMMuxCxe7fapnJfXltQD6gt6YG8jVBkocvU3MgMqcctlWnibpTCBlBPATnWe8t6eYCq0UplLIoaOPmUwIyX4OsWL2dIicdEE/640?wx_fmt=jpeg)

  

飞书的语音转文字功能特别好用，在私信聊天对话框中，也支持机器人生成各种文档、图片，直接发给你，然后飞书的聊天框也可以分门别类地把你的文件、聊天记录用文档方便地查阅。

  

我比较常用的功能之一是把公众号的文章，甚至是其他平台的链接直接丢给飞书的这个机器人。

  

然后这个机器人不仅会记住，而且还和我会深度沟通，并且把我认为重要的任务单独更新到文档里，以备之后随时可以查阅。

  

飞书的私信体验太好了，找东西不像微信那么难找，而且真的是一个很全能的入口，用过电报机器人的同学都懂我在说什么，工作效率的提升还得靠飞书啊。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/7SRWibfxWe0EIvEEU4oL315jP9t60ichsufwxACRYosfmic7BZA7m0oRg0GDllVSCqBWicJZuuz4YAlwJpUibMxNAJK3BHP0iaibvN3PMz4ObSiaBIo/640?wx_fmt=jpeg)

  

这个是我第二个飞书机器人，它的后台来源于我电脑端的应用，配合谷歌反重力一起开发的内容系统，实现效果见[「星佳数字生态系统 V1.5」](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649832008&idx=3&sn=759ab60855c1e6fb1a208e2b1e8dfdbb&scene=21#wechat_redirect)

  

也就是说，只要我的服务器程序电脑开机并且运行了内容系统的程序，飞书是可以直接帮我交互用指令 /xhs  交互返回内容的。

  

根据我几百万字内容训练的文案，能生成专属星佳风格的短内容，并且输出对应的图片直接到飞书，方便手工粘贴复制到各大平台。

  

别试图绕过平台的风控批量去发，封号就是最终的结局！

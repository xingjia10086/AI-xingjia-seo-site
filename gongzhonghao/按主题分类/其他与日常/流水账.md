流水账
===

原创 星佳 星佳 [星佳是个小人物](javascript:void\(0\);)

在小说阅读器中沉浸阅读

昨天的航班延误，从首都机场出来已经快2 点，叫车到住处，收拾完三点多了，居然完全没有任何困意。

  

北京的夜风吹着我的拖鞋和背包，清醒的疯狂链接各种神经元！

  

不知道哪门子抽风，突然想起一句歌词，来自 Jien「 现在艺名叫 Tank ，三国恋，非你莫属等等都是他的原创！」

  

 凌晨的天空总是容易想念

 还来不及和你说一声抱歉

 对不起 我的 baby

  

这首歌只是 Demo，并没有公开发行过，是星佳初中的时候在 163888 上听了很多遍的一个早期歌单。

  

那个网络歌手最早的 ID 叫 零，之后改成了 Jien，甚至为了姐妹甜甜，写了首说唱 Diss 抄袭的那个家伙，而他翻唱的半岛铁盒，至今也是忘不掉的味道！

  

不知道还有多少人记得那个时代，互联网刚开始可以流畅的听音乐，编曲和录音软件的门槛降低到人人都可以写原创音乐，于是诞生了 QQ 音乐四巨头，也有了赵英俊的成名曲「刺激 2005」

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tOFSAibJ5wqu82meibGZ9tlknkJibVeZNMbXibadfdagrhI3sC3IZicGLVTOBicDhw5Dd3A9y5pBicoEgdKKYXnCw0gkA/640?wx_fmt=jpeg)

  

而 20 年后，赵英俊已经鼎鼎大名成为电影圈首屈一指的配乐大师，强烈推荐「送你一朵小红花」和「火锅英雄」电影原声带！

  

但他本人已不在，留下了很多优秀的音乐作品，和一些精彩的电影角色。

  

消失很久的 Tank 也回来了，在杭州成功做了心脏手术，虽然因为家族的先天性心脏病失去了姐姐，也曾经沉溺酒精自暴自弃了很久，最终因为教会的帮助终于走了出来。

  

还有很多那个时代的音乐人的作品陪着星佳从少年到如今的中年，昨天为了找一首歌，翻遍了整个互联网，好在 Street Voice 和 Youtube 有挺多热心人的备份。

  

于是，可以回答今天朋友圈发的这个问题了。

  

一个中年男人不抽烟不喝酒不赌博不玩游戏，甚至驾照都没有，不玩摄影不懂表压根也不去钓鱼，你们说，他活着图啥？\[破涕为笑\]

  

这个男人喜欢写东西，而且喜欢夜里写乱七八糟的流水账，同时还时不时就要听一听20 年前的老歌，和 MV。

  

写完文章会习惯一个人打开投影看电影，多半看到某些桥段又会触景生情，泪眼婆娑，直到咬着被子小声大哭。

  

不管写文章，还是看电影，都是一种精神享受和情绪输出的爱好，的确有研究发现，一个人如果持续写日记，并且时不时分泌泪液，的确活的会久一点儿的。

  

还记得前一阵随手打开马丽主演的「水饺皇后」，而且之前还大概查了下创始人的真实故事，老实说，电影改编的还是太美好了，真实的人生经历如果拍出来一定会有观众骂编剧瞎写了。

  

就是这么一部电影，看的过程中数次落泪，不知道到底是因为主角的不幸，还是投射到了自己的幻觉，好在电影总会结束，梦总会醒来。

  

还有贾玲那部瘦身电影也是，甚至去完整的看了记录电影拍摄过程的纪录片。

  

电影院里没尽兴，又等到「破地狱加长版」虽然只增加了十多分钟的内容，但哭的更是昏天暗地。

  

你还别说，那天和某个同路人录播客，通过互联网两个人对着麦克风聊天，突然间提到大半夜有个 Github 上的网友无偿又热心的帮着解决问题，突然间就开始忍不住哭起来了，可能是对开源精神的赞颂，也可能是对于互联网上陌生人无私帮助的感激，又可能是真实社会和虚拟互联网上的冰冷对比的寒心吧。

  

又流水线瞎写了这么多，但确定的是，这次独自一人来北京，似乎有点喜欢北京了，但我知道，北京还是那个北京，变的只是自己而已！

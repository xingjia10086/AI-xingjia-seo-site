时不时给孩子们添乱的父母
============

原创 星佳 星佳 [星佳是个小人物](javascript:void\(0\);)

在小说阅读器中沉浸阅读

今天听一个投资人朋友讲了挺多悲伤的故事，尤其是他总会习惯的说「农夫与蛇」这个典故，大意就是看好某个年轻人，于是给年轻人投了第一笔钱，当然他也做好了最坏结果的准备就是全部资金打水漂。

但尴尬的是，年轻人做了一阵子业务后，希望大哥继续投钱，不然项目很难继续下去，但各种原因大哥的确没办法再投入，于是弄的不欢而散，成为形同陌路的仇人。

我跟大哥说，其实 农夫与蛇 这个典故呢，用科学角度看，其实和善良无关，因为对于蛇来说，之所以它叫冷血动物，是因为它的脑结构就没有情感一说，抵御危险才是他的正常反应，所以只能怪农夫活该，吃了没文化的亏而丧命。

为了安慰大哥，我也说了一些更惨的事，之后彼此卖卖惨，但过去的事就让他们随风而去，未来还有很多机会，AI Web3 和 机器人等等，核心还是找到靠谱的年轻人才行，于是我们的共识还是去给顶级高校和教授，理工科的学生们投钱，而不是在社会上摸爬滚打的老创业者撒钱，至于原因，就不多说了。

前阵子试用了阿里新出的AI编程产品 心流，类似Qwen 和 Claude Code 的命令行工具，当然也提供了给普通用户的网页版。

> 心流（iFlow）是由阿里巴巴旗下杭州万相创意科技有限公司开发的一款AI搜索助手，旨在通过智能技术提升用户在科研、教育、职场等场景中的知识获取和工作效率。它基于淘宝星辰大模型，具备强大的长文本处理能力，能够快速检索、分析学术论文等复杂文档，准确率高达99%以上。

  

然后用深度搜索查了下 沉浸式翻译 这款一人开发应用的历程，居然又是腾讯的同事做的，起初就一个人开发到获得百万用户，直到被某公司收购，其实翻译需求一直以来都是大公司霸占的热门领域，在AI时代，居然一个人开发的产品成了上千万用户的选择，这背后能给我们很多启发。

所谓选择大于努力的意思就是，想清楚到底要把自己有限的青春放到什么地方去贡献，这是至关重要的事。

和父母教育孩子一样，我完全不相信厉害的孩子真的不被父母规划，不被父母引导，真的靠野蛮生长就能成为天子骄子，研究犹太人如何教育孩子的父母都知道，原本我们以为的藤校不过是人家用来培养员工的地方而已，他们的后代接受的教育或者学校的名字，你是压根很难靠搜索找到信息的，甚至想入读他们的学校，不止要有钱，还得有人推荐。

之前研究过一个问题，为何犹太人这么一个历史上太多悲剧的民族，居然如此厉害，不管是财阀家族，还是诺贝尔奖得主，总是少不了他们的身影，直到听到一个解释，最初想成为犹太教只有一个条件，得识字，于是那个时代不认识字的孩子和家庭压根不可能加入犹太教。

很多我们想不明白的事，通过AI的深度研究，其实都可以找到一个可能的答案，而这个答案或许就可以启发你的人生。

关于培养孩子，最怕父母随波逐流，于是小星星幼儿园大班就开始用平板，也早就通关了 switch 版的超级玛丽兄弟，直到最近把平板没收，开始送给他用笔记本电脑，好在孩子已经识字，处于强烈的好奇心阶段，如今使用触控板和键盘，刚入门电脑使用，至于为何要让孩子多用电脑，少用平板可以看看文章 [为人父母，别再骗孩子们手机是爸爸妈妈用来工作的生产力工具了！](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649831476&idx=1&sn=1707aed06b62a190bdb1cf3561f9f52f&scene=21#wechat_redirect)

那么多兴趣班，最后剩下的就是动手和电路板打交道，每周两节课但暑假后发现，除了课上之外，小伙子回家完全不碰这些东西，于是今天跟他商量，如果真的不喜欢，所以别去学了，爸爸和老师说下，剩下的课退费即可。

臭小子极度不乐意，目的就是为了自己亲手可以做一个游戏机，其实稍微懂编程的人直到做个红白机那种的掌机其实不难，开源硬件烧录点软件就好，有这个目标也挺好，于是为了考察他的学习进度，我把他带回家的成品拆了，让他重新组装一次。

折腾了十多分钟，的确把主板，屏幕和控制器连接后，点亮了显示器，但问题是输出的数据不对，压根没获取到传感器的实时数据，于是我们想了几个可能性，计划明天用电脑连接电路板，看看是不是控制代码的问题。

一旁刚吃完水果的弟弟也来凑热闹，于是给兄弟俩看了看那个大哥最新投资的机器人演示视频，两个孩子很喜欢，一直问各种问题，还记得有一次，让老二去收拾自己的玩具，他居然脱口而出，为什么爸爸不买个机器人帮我收拾玩具啊，为什么要我去收拾玩具啊，我好累啊！！

相信过不了多久，家庭组装机器人也不是梦，开源组件越来越便宜，在AI的协助下，只要你想做个东西，基本上和AI一直互动下去，肯定能做出来个能用的DEMO产品的，这也是沉浸式翻译作者的愿景，让知识平权，而不是被语言限制。

而和孩子们最好的亲子陪伴，除了去户外疯跑之外，就是和孩子们一起动手做一个他们想做的东西，最好需求是他们提的，比如陪他们睡前讲故事，还可以聊天的智能AI音箱，然后理工科爸爸想办法和孩子们一起学习，一起动手，之后一起完成私人定制的产品给自己用，接下来在使用过程中继续迭代，微创新，然后找到更多目标用户就好。

往往很多伟大的idea，或者产品需求都来自不懂技术，也不懂互联网的普通人，而孩子们是最不会说客套话的群体，所以为人父母，能从小和孩子们培养动手能力，一起研究和开发解决他们问题的产品，这种财富和智慧才是能陪伴一生的东西。

## 📂 仓库里都有些什么宝藏？

这个仓库详尽地记录了星佳在不同维度、不同平台上的成长历程、深度思考与生活点滴。这里所有的内容，都是时间与阅历最真实的见证：

- 📝 **微信公众号深度长文 (`/gongzhonghao/`)**：涵盖了几千篇公众号推文的沉淀。这里有关于创业、前沿技术探索、生活随笔以及个人成长的系统化思考。
- 📸 **十几年的朋友圈原貌 (`2012年.md` - `2026年.md`)**：以年度为单位，详尽记录了从 2012 年至今跨越十几年的朋友圈动态。每天的碎片化灵感、最真实的喜怒哀乐，以及那些不经意间闪光的时刻，都被原封不动地保留了下来。
- 🌐 **微博视角的时代观察 (`/weibo/`)**：记录了那些简短却一针见血的观点，以及多年来对时代脉搏和行业变迁的实时捕捉。
- 🤖 **AI 赋能的自我复盘 (`/almanack_data/`, `*.js`)**：仓库里不仅有文字档案，更包含了大量前沿的 Node.js 脚本工程（如 `generate-chapters.js`, `extract-quotes.js` 等）。这是我在尝试通过 AI 技术，分析多平台海量语料，从中提取精华并自动编写《星佳年鉴》的勇敢探索。

这里的每一个字节，都是时光在一个人身上雕刻出的独特印记。
